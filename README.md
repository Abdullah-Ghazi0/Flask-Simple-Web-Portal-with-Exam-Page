# Flask-Portal-with-Exam-Meme-Game-Messaging-system
A beginner-friendly Flask web portal where users can register, log in, and access a dashboard with a simple exam system and meme page, it also have a simple hangman game and messaging system

## Users can:
* Register and log in
* Change their password
* Delete their account
* Take an exam created by the admin
* View Memes
* Play Hangman
* Message other users

## Admins can:
* Add exam questions to the database
* Add words to Hangman Game

This project is intended for learning purposes and demonstrates basic API, authentication and database operations using Flask.

## Features:
* User registration and login system
* Passwords are securely hashed before storing in the database
* Role-based access (admin and users)
* Users can change their password or delete their account
* Admin can add exam questions
* Users can take exams
* Exam results are stored in the database
* Users can also view Memes fetched from reddit using API
* Admin can add words to Hangman game
* Users can play hangman
* Users can message other users


## Technologies Used:
- Python (Flask)
- SQLite
- flask_sqlalchemy
- SQLAlchemy
- Werkzeug
- HTML
- Jinja2
- Requests

## Python Version
This project is made in python 3.14

## How to Run the Project:

1. Clone the repository
2. Install required dependencies: (pip install -r requirements.txt)
3. Run the application: (python run.py)
4. Open your browser and visit: (http://localhost:5000)
